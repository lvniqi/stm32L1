Update History

  

V1.1.0 / 24-February-2014

 

Main Changes

•  Use latest STMTouch Driver.

 

Content

•  STM32L152_Ex01_3TKeys_EVAL: Example with 3 TKeys (3 Channels / 3 Banks) on Eval board.

•  STM32L152_Ex02_Linear_EVAL: Example with 1 Linear sensor (3 Channels / 3 Banks) on Eval board.

•  STM32L152_Ex03_10TKeys_Lin_Rot_TS_EVAL: Example with 10 TKeys + 2 LinRot sensors on Sensor board.

•  STM32L152_Ex04_LPM_3TKeys_EVAL: Example with 3 TKeys (3 Channels / 3 Banks) and Low Power on Eval board.

•  STM32L152_Ex05_2TKeys_EVAL: Example with 2 TKeys (2 Channels / 2 Banks) on Eval board.

•  STM32L152_Ex06_Linear_EVAL: Example with 1 Linear sensor (3 Channels / 1 Bank) on Discovery board.

•  STM32L152_Ex07_10TKeys_Lin_Rot_TS_EVAL: Example with 10 TKeys + 2 LinRot sensors on Sensor board.

•  STM32L152_Ex08_LPM_3TKeys_EVAL: Example with 2 TKeys (2 Channels / 2 Banks) and Low Power on Eval board.

•  STM32L152_Ex09_2TKeys_EVAL: Example with 2 TKeys (2 Channels / 2 Banks) on Eval board.

•  STM32L152_Ex10_Linear_EVAL: Example with 1 Linear sensor (3 Channels / 1 Bank) on Discovery board.

 

Development Toolchains and Compilers

•  IAR Embedded Workbench for ARM V6.70.2.

•  KEIL MDK-ARM V5.01.

•  ALTIUM TASKING V4.3r1.

•  ATOLLIC TrueSTUDIO V4.3.1.

•  RAISONANCE Ride7 V7.48 + RKitARM V1.52.

 

Supported Devices and EVAL boards

•  STM32L152D-EVAL Board (MB964 rev B).

•  STM32L Controller Board (MCD10-021 rev B) + STM32L Sensor Board 1 (MCD10-022 rev B).

•  STM32L152-EVAL (MB819 rev B).

•  STM32L152-Discovery (MB963 rev B).

•  STM32L152-EVAL (MB819 rev A).

 

Known Limitations

•  None.
